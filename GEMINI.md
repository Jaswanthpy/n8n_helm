# n8n Kubernetes Deployment

This project focuses on hosting n8n in Kubernetes using Helm charts in a new namespace.

## Useful Links

- [n8n Hosting Documentation](https://docs.n8n.io/hosting/) - Official documentation for hosting n8n
- [n8n Helm Chart](https://artifacthub.io/packages/helm/community-charts/n8n) - Community Helm chart for deploying n8n on Kubernetes

## Project Overview

The goal is to deploy n8n (workflow automation tool) in a Kubernetes cluster using Helm charts, creating a dedicated namespace for the deployment.
